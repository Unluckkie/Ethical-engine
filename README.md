# Ethical-engine
A complementary visualizer for my ethical framework using simple python 

https://ethical-engine-gnyfp6q9yiv2jddpmrxaqz.streamlit.app/

Link to use it but you can run it yourself using the python script and the requirements here

Basically tldr: it's basically a visualizer that can serve to visualize the framework for my ethical system focusing on 4 equally weighted quadrants

Quadrant 1: Environmental and material means of the individual/variable
Quadrant 2: The rights of the individual or variable as defined by the law, or can be universally applied to universal laws on equity and equality
Quadrant 3: The net societal P&L of an action 
Quadrant 4: The laws as a backbone, but can be veto'd if quadrant 4 is outnumbered.

This script basically contains a resolution in case of a TIE in ethical decisions utilizing variable amplitude.
